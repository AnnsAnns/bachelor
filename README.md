# Porting RIOT OS to the RP2350: An Exploration of a Heterogeneous Architecture and Programmable I/O

This is a public mirror of my bachelor thesis "Porting RIOT OS to the RP2350: An Exploration of a Heterogeneous Architecture and Programmable I/O". The original thesis can be found [here](https://inet.haw-hamburg.de/thesis/completed/ba_hert_tom.pdf/view).

The thesis uses Typst and was mostly developed using Tinymist, though it should be possible to compile it with any Typst installation.
